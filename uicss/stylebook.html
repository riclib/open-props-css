Utility Classes

Spacing Utilities

Margin and padding utilities are available from 0-4:

  • .m-0 to .m-4 - All sides margin
  • .mt-0 to .mt-4 - Top margin
  • .mb-0 to .mb-4 - Bottom margin
  • .p-0 to .p-4 - All sides padding
  • .px-1 to .px-4 - Horizontal padding
  • .py-1 to .py-4 - Vertical padding

Display Utilities

  • .hidden - Hide element
  • .block - Display block
  • .inline - Display inline
  • .flex - Display flex
  • .grid - Display grid

Border & Shadow

  • .border - Add border
  • .border-0 - Remove border
  • .rounded - Rounded corners
  • .rounded-full - Fully rounded
  • .shadow - Box shadow
  • .shadow-lg - Large box shadow

Other Utilities

  • .w-full - Width 100%
  • .cursor-pointer - Pointer cursor
  • .select-none - Disable text selection
  • .sr-only - Screen reader only

Utility Code Examples

<!-- Spacing -->
<div class="mt-4 mb-2 px-3">Spacing example</div>

<!-- Flexbox with gap -->
<div class="flex gap-2 items-center justify-between">
    <span>Left</span>
    <span>Right</span>
</div>

<!-- Border and shadow -->
<div class="border rounded shadow p-3">
    Card with border and shadow
</div>

<!-- Combining utilities -->
<button class="w-full mt-4 rounded-full shadow-lg">
    Full width button
</button>

Customization Code Examples

<!-- Custom colored buttons -->
<button style="background-color: var(--purple-6); border-color: var(--purple-7);">
    Purple Button
</button>

<!-- Gradient backgrounds -->
<div class="card" style="background: var(--gradient-15);">
    Gradient card content
</div>

<!-- Custom spacing using Open Props sizes -->
<div style="padding: var(--size-7); margin-block: var(--size-5);">
    Content with custom spacing
</div>

<!-- Typography with Open Props -->
<p style="font-size: var(--font-size-5); 
          line-height: var(--font-lineheight-1);
          font-weight: var(--font-weight-7);">
    Large bold text
</p>

<!-- Animations -->
<button style="animation: var(--animation-pulse);">
    Animated button
</button>

<!-- Custom shadows -->
<div style="box-shadow: var(--shadow-4);">
    Deep shadow box
</div>

<!-- Using Open Props colors -->
<div style="background-color: var(--indigo-1); 
            color: var(--indigo-9);
            border: 2px solid var(--indigo-3);">
    Indigo themed box
</div>

<!-- Combining multiple properties -->
<div style="background: var(--gradient-22);
            padding: var(--size-6);
            border-radius: var(--radius-3);
            box-shadow: var(--shadow-3);
            animation: var(--animation-fade-in);">
    Fully customized element
</div>
